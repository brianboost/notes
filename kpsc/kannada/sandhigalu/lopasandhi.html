📘 ಸಂಧಿಗಳು

ನಾವು ಮಾತನಾಡುವಾಗ ಕೆಲವು ಶಬ್ದಗಳನ್ನು ಬಿಡಿಬಿಡಿಯಾಗಿ ಹೇಳುವುದಿಲ್ಲ. ಅವುಗಳನ್ನು ಕೂಡಿಸಿ ಹೇಳುವುದೇ ಸಂಧಿ.

🔹 ಸಂಧಿ ಎಂದರೇನು?

ಎರಡು ಶಬ್ದಗಳು ಅಥವಾ ಅಕ್ಷರಗಳು ಕಾಲವಿಳಂಬವಿಲ್ಲದೆ ಸೇರುವುದು ಸಂಧಿ.

📊 ಪ್ರಕೃತಿ + ಪ್ರತ್ಯಯ
ಪ್ರಕೃತಿ ಪ್ರತ್ಯಯ ರೂಪ
ಆಡು ಇಸು ಆಡಿಸು
ಮರ ಅನ್ನು ಮರವನ್ನು
ಪುಸ್ತಕ ಇಂದ ಪುಸ್ತಕದಿಂದ
ದೇವರು ಇಗೆ ದೇವರಿಗೆ
📊 ಪದ + ಪದ
ಕೂಡಿಸಿದ ರೂಪ ಕೂಡಿಸದ ರೂಪ
ಅವನಂಗಡಿ ಅವನ ಅಂಗಡಿ
ಅವನಿಗಿಲ್ಲ ಅವನಿಗೆ ಇಲ್ಲ
ಹಣ್ಣಿನಂಗಡಿ ಹಣ್ಣಿನ ಅಂಗಡಿ
📌 ಸಂಧಿಯ ವಿಧಗಳು

(i) ಸ್ವರ + ಸ್ವರ → ಸ್ವರಸಂಧಿ

(ii) ವ್ಯಂಜನ → ವ್ಯಂಜನಸಂಧಿ

(iii) ಬದಲಾವಣೆಗಳು → ಸಂಧಿಕಾರ್ಯಗಳು

📘 ಲೋಪಸಂಧಿ

ಎರಡು ಸ್ವರಗಳು ಸೇರಿ ಮೊದಲನೆಯ ಸ್ವರ ಲೋಪವಾಗುವುದು.

📝 ಉದಾಹರಣೆಗಳು

ಊರು + ಅಲ್ಲಿ = ಊರಲ್ಲಿ

ಮಾತು + ಇಲ್ಲ = ಮಾತಿಲ್ಲ

ಆಡು + ಇಸು = ಆಡಿಸು

ಬೇರೆ + ಒಬ್ಬ = ಬೇರೊಬ್ಬ

ನಾವು + ಎಲ್ಲಾ = ನಾವೆಲ್ಲಾ

📌 ನಿಯಮ

ಅರ್ಥ ಕೆಡದಿದ್ದರೆ ಮಾತ್ರ ಸ್ವರ ಲೋಪವಾಗುತ್ತದೆ.

⚠️ ಗಮನಿಸಿ

ಮನೆ + ಇಂದ → ಮನಿಂದ ❌ (ತಪ್ಪು)

ಗುರು + ಅನ್ನು → ಗುರನ್ನು ❌ (ತಪ್ಪು)

✅ ಸಾರಾಂಶ

✔ ಎರಡು ಅಕ್ಷರಗಳು ಸೇರುತ್ತವೆ

✔ ಸ್ವರ ಲೋಪವಾಗಬಹುದು

✔ ಅರ್ಥ ಕೆಡಬಾರದು
